프로토타입의 기획을 기준으로, 실제 저장소에 해당 기능이 구현되어 있는지 조사한 결과입니다.

4. 발견된 이슈

CRITICAL 외부 이미지 URL 의존 (raw.githubusercontent.com)

프로토타입에서 GitHub Raw URL 사용 — 프로덕션에서는 속도/정책 이슈 발생.

권장 수정: CDN(CloudFront, 사내 이미지 서버)으로 전환 + asset 경로 통일.

WARNING 타이머/애니메이션 클린업 미흡

페이지 전환 시 setTimeout/requestAnimationFrame 정리 누락 — 메모리 누수 가능.

권장 수정: useEffect return 절에서 clearTimeout, cancelAnimationFrame 처리.

IMPROVEMENT 메타데이터 하드코딩 (사용자명/분수/단어수)

프로토타입 단계라 고정값 사용 중.

권장 수정: props 또는 React Query로 API 연동.
