FL 리포트 v3 (FLReport3)

카테고리: 리포트 적용 범위: 한국, 일본 대상: FL 수업 완료자 데이터: AI 분석 결과 + 공인 영어 레벨
개요: 오각형 차트 + 레벨 성장 예측 로직 강화 버전
FL 수업 완료 후 제공되는 AI 기반 수업 분석 리포트. 5영역 오각형 차트, 고급 어휘 추천, 말하기 습관, 문장 교정, 레벨 성장 예측을 포함합니다.

1. 주요 기능

  • 인사말 + 영어 레벨 카드 (이모지/레벨명/설명)
  • 5영역 오각형 차트 (의사소통/어휘/문법/유창성/이해도)
  • 어휘 난이도 비중 & 고급 어휘 추천
  • 말하기 습관 피드백 (3가지)
  • 문장 교정 연습 (3~5문장)
  • 3개월 후 레벨 성장 예측 (1~6:+2, 7~9:+1, 10:최고)

4. 발견된 이슈

WARNING Inline 스타일 과다 (900줄+)

리포트 렌더링 코드에 inline style이 섞여 유지보수 난이도 상승.

권장 수정: 외부 SCSS 모듈로 분리 + 디자인 토큰 적용.

IMPROVEMENT 레벨 성장 로직 스펙 vs 구현 검증

스펙: 1-6 +2, 7-9 +1, 10 최고 — 실제 구현과 일치하는지 API 레벨 응답 검증 필요.

권장 수정: 단위 테스트로 10개 레벨별 예상치 검증.

IMPROVEMENT 오각형 차트 null 처리

5영역 중 일부 점수가 null일 때 평균값 없으면 렌더링 오류.

권장 수정: 값 미제공 시 점을 중앙으로 두고 "데이터 부족" 레이블 노출.
