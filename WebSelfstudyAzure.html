웹 스스로학습 Azure 발음평가 (WebSelfstudyAzure)

카테고리: 학습 적용 범위: 한국, 일본 대상: 웹 강의실 스스로학습 사용자 데이터: Azure 발음 평가 결과
개요: Azure 기반 발음 점수 + 4메트릭 시각 피드백
웹 강의실 스스로학습의 Say-the-Sentence 단계에서 사용자 녹음을 Azure Speech Services로 평가하고 점수/메트릭/단어별 색상 코딩/음절 분석을 제공합니다.

  • 문장 재생 + 사용자 녹음 + 점수 피드백
