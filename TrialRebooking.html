무료체험 결석자 재신청 (TrialRebooking)

카테고리: 체험 적용 범위: 전체 대상: 무료체험 최초 1회 결석자 (연속 2회 제외) 데이터: 결석 수업 정보 + 추천 슬롯 4개
개요: 첫 결석 회원의 빠른 재신청 유도 팝업
무료체험 1회 결석자에게 원래 신청 시간 기반 슬롯 추천 팝업을 노출합니다. 2회 연속 결석자는 제외하며, "오늘 그만 보기" / "X 닫기" 2가지 닫기 정책을 사용합니다.

1. 주요 기능

  • Bottom Sheet 팝업 + 닫기 2종(오늘 그만 보기 / X)
  • 슬롯 4개 추천 (전화/화상 + 10/20분 유지)
  • 알고리즘: 원래 시간 → 인접 시간(+10→-10→+20→-20) → 내일로 전환
  • "추천" 배지 (원래 또는 인접 시간)
  • 컨펌 팝업 (다시 선택 / 확인)

4. 발견된 이슈

CRITICAL 슬롯 추천 알고리즘 프론트 구현 — 백엔드 계산 권장

Step 1~5 알고리즘이 프론트에서 실행되면 타이밍/정확도 문제. DB 쿼리가 필요.

권장 수정: POST /trial/rebooking/slots 에서 서버가 4개 슬롯을 반환하도록.

WARNING "오늘 그만 보기" 24시간 기준 모호

자정(00:00)인지 24시간 후인지 타임존 이슈.

권장 수정: 자정(KST/JST) 기준으로 확정 + expireAt을 서버가 내려줌.

IMPROVEMENT [다른 시간 보기] 확장 후 슬롯 개수 미정

스크롤 전략/페이징 규칙 부재.

권장 수정: 하루 20개 슬롯까지 가상 스크롤로 렌더.
