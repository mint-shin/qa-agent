수업 평가 팝업 (LessonUxData)

카테고리: 수업 적용 범위: 한국, 일본 대상: 영어/제2외국어/전략어 수강자 데이터: 수업/강사/교재/환경 평가
개요: 수업 종료 후 강사/교재/환경 설문 수집
수업 종료 직후(또는 홈/강의실에서 재진입 시) 팝업으로 띄워지는 설문입니다. 전체 수업 평가 + 강사/교재 별점 + 수업 환경 불편 사항을 한 번에 수집합니다.

1. 주요 기능

  • 전체 수업 평가 3선택 (아쉬워요/괜찮았어요/좋았어요)
  • "좋았어요" 선택 시 Confetti 애니메이션
  • 강사·교재 별점 (1~5), 낮은 점수 시 아쉬운 점 다중선택
  • 전략어/프리토킹 수업은 교재 평가 미노출
  • 수업 환경 평가 (화상 전용 항목 분기)
  • 제출 후 결과 팝업 문구 동적 변경

4. 발견된 이슈

CRITICAL 전략어/프리토킹 판별 로직 미정의

교재 평가 영역을 숨겨야 하는데 플래그 기준이 LMS인지 수업 메타인지 명확하지 않음.

권장 수정: lesson.type === "STRATEGIC" || lesson.type === "FREE_TALKING" 기준으로 통일.

WARNING 자동 스크롤 ref 관리 복잡 (5개 섹션)

teacher/content/env/praise/etc ref가 동시에 관리되어 버그 소지 높음.

권장 수정: ref Map 또는 IntersectionObserver로 단일화.

IMPROVEMENT Byte 계산이 한글 2byte 가정

UTF-8 실환경에서 한글 3byte가 대부분 — 100byte 제한이 의도와 다를 수 있음.

권장 수정: TextEncoder로 실제 UTF-8 byte 계산.
