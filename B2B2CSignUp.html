B2B2C 회원가입 (B2B2CSignUp)

카테고리: 가입 적용 범위: 한국 대상: B2B2C 제휴 기업 회원 데이터: 제휴 코드 + 회원 기본 정보
개요: 제휴코드 기반 가입 경로 강화 + 소셜 로그인
기업 제휴 코드를 받은 회원을 위한 가입 흐름. 코드 유형에 따라 B2B2C(혜택 안내) 또는 B2B(확장 폼)로 분기하며, 카카오/네이버 소셜 로그인을 기본으로 제공합니다.

1. 주요 기능

  • Join Main (카카오/네이버 + 기업회원 가입 링크)
  • Company Code 입력 → B2B2C / B2B 분기
  • Join Benefit (할인율/제휴사명 표시)
  • B2B 조건부 폼 (이름/전화/사번/부서/직급)

프로토타입의 기획을 기준으로, 실제 저장소에 해당 기능이 구현되어 있는지 조사한 결과입니다.

4. 발견된 이슈

CRITICAL 제휴코드 로컬 저장(1일) 만료 처리 미구현

24시간 후 만료돼야 하는데 타이머/expire 로직이 없음.

권장 수정: localStorage에 {code, expireAt} 저장, 읽을 때 현재시간 비교.

WARNING B2B 확장 폼 애니메이션/스크롤 가이드 부재

폼이 펼쳐질 때 자동 스크롤이 안 되면 사용자가 입력 필드를 찾지 못함.

권장 수정: scrollIntoView + 진입 애니메이션 (height 300ms).

IMPROVEMENT 네이버 가입 예외 케이스 안내 부실

네이버 로그인은 회원가입 전 얼럿을 띄울 수 없는 기술 제약이 있는데, 사용자 공지가 명확하지 않음.

권장 수정: 가입 완료 후 최초 로그인 시 "제휴코드 등록" 가이드 토스트.
