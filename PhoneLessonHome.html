전화영어 홈 대시보드 (PhoneLessonHome)

카테고리: 홈 적용 범위: 전체 대상: 수강 중(B2C/B2B2C) 데이터: 다음 수업 + 통계 + 강사 리스트
개요: 활성 수강자의 메인 홈
수강 중인 회원 대상 홈 화면. 다음 수업 카운트다운, 학습 통계, 추천 강사, 빠른 예약 등 핵심 기능을 한 화면에 제공합니다.

  • 6. 권장 로드맵

1. 주요 기능

  • 상단 헤더 (로고/알림/설정)
  • 다음 수업 카드 (강사명/시간/분수/HH:MM:SS 카운트다운)
  • 학습 통계 (진도율, 연속 학습일, 누적 분)
  • 추천 강사 리스트 (평점, 리뷰, 전문 분야, 가용 슬롯)
  • 빠른 예약 (시간 → 강사 → 예약)

2. 데이터 모델

필드설명
MOCK_USERname, level, levelDesc

MOCK_STATScompletedCount, targetCount, streakDays, totalMinutes
MOCK_TEACHERS[]name, rating, reviewCount, specialties, availableSlots

3. 크로스레퍼런스 — 저장소별 구현 상태

프로토타입의 기획을 기준으로, 실제 저장소에 해당 기능이 구현되어 있는지 조사한 결과입니다.

저장소상태비고
Uphone3_front부분기존 홈 컴포넌트 존재 — 교체 or 리뉴얼 필요
uphone3_web없음웹 홈과 차이점 확인 필요
uphone3_api부분스케줄/강사 API 존재
uphone3_lms부분강사 정보·시간대 관리
